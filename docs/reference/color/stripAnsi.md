# stripAnsi

Removes all ANSI color/style codes from a string.

```typescript
import { color, stripAnsi } from 'es-toolkit/color';

const colored = color.red('hello');
stripAnsi(colored);
// Returns: 'hello'
```

## Usage

### `stripAnsi(text)`

Removes ANSI escape codes from a colored string and returns the plain text. Useful when saving logs to a file or measuring the visible length of a string.

```typescript
import { color, stripAnsi } from 'es-toolkit/color';

const message = color.bold(color.red('Error occurred'));

// Remove ANSI codes when saving to a file
fs.writeFileSync('log.txt', stripAnsi(message));

// Measure string length
const visibleLength = stripAnsi(message).length;
```

It removes all SGR (Select Graphic Rendition) sequences generated by this library, including basic colors, 256 colors, and RGB.

#### Parameters

- `text` (`string`): A string that may contain ANSI codes.

#### Returns

(`string`): The string with all ANSI codes removed.
