# stripColor

Removes ANSI color and style codes from a string.

```typescript
import { color, stripColor } from 'es-toolkit/color';

const colored = color.red('hello');
stripColor(colored);
// Returns: 'hello'
```

## Usage

### `stripColor(text)`

Removes ANSI escape codes from a colored string and returns the plain text. Useful when saving logs to a file or measuring the visible length of a string.

```typescript
import { color, stripColor } from 'es-toolkit/color';

const message = color.bold(color.red('Error occurred'));

// Remove ANSI codes when saving to a file
fs.writeFileSync('log.txt', stripColor(message));

// Measure string length
const visibleLength = stripColor(message).length;
```

It removes SGR (Select Graphic Rendition) sequences generated by this library, including basic colors, 256 colors, and RGB. It does not strip cursor movement, OSC hyperlinks, or other non-SGR sequences.

#### Parameters

- `text` (`string`): A string that may contain ANSI codes.

#### Returns

(`string`): The string with all ANSI codes removed.
