# stubTrue

::: info
出于兼容性原因，此函数仅在 `es-toolkit/compat` 中提供。它可能具有替代的原生 JavaScript API，或者尚未完全优化。

从 `es-toolkit/compat` 导入时，它的行为与 lodash 完全一致，并提供相同的功能，详情请见 [这里](../../../compatibility.md)。
:::

返回 `true`。

## 签名

```typescript
function stubTrue(): true;
```

### 返回值

(`true`): 始终返回`true`。

## 示例

```typescript
stubTrue(); // Returns true
```
