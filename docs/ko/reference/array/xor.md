# xor

두 배열 간의 대칭 차집합을 계산해요. 대칭 차집합은 두 배열 중 하나에는 있지만 교집합에는 없는 요소들의 집합이에요.

## 인터페이스

```typescript
function xor<T>(arr1: T[], arr2: T[]): T[];
```

### 파라미터

- `arr1` (`T[]`): 첫 번째 배열이에요.
- `arr2` (`T[]`): 두 번째 배열이에요.

### 반환 값

(`T[]`): `arr1` 또는 `arr2`의 요소들 중 두 배열의 교집합엔 없는 요소들을 포함하는 배열이에요. (대칭 차집합)

## 예시

```typescript
// [1, 2, 5, 6]을 반환해요.
xor([1, 2, 3, 4], [3, 4, 5, 6]);

// ['a', 'c']을 반환해요.
xor(['a', 'b'], ['b', 'c']);
```
