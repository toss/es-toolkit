Node.js나 Bun을 사용하는 경우, [npm](https://npmjs.com/package/es-toolkit)에서 설치할 수 있어요. Deno에서는 [JSR](https://jsr.io/@es-toolkit/es-toolkit)에서 설치할 수 있어요.

브라우저에서 바로 쓰는 경우, CDN에서 가져올 수 있어요.

## Node.js & Bun

es-toolkit은 Node.js 18 또는 이후 버전과 Bun을 지원해요. es-toolkit을 설치하기 위해서는 아래 명령어를 사용해주세요.

<Tabs items={['npm', 'pnpm', 'yarn', 'bun', 'deno']}>
  <Tab value="npm">
    ```sh
    npm install es-toolkit
    ```
  </Tab>
  <Tab value="pnpm">
    ```sh
    pnpm add es-toolkit
    ```
  </Tab>
  <Tab value="yarn">
    ```sh
    yarn add es-toolkit
    ```
  </Tab>
  <Tab value="bun">
    ```sh
    bun add es-toolkit
    ```
  </Tab>
  <Tab value="deno">
    ```sh
    deno add @es-toolkit/es-toolkit
    ```

    Deno에서 사용하면, JSR에서의 제한으로 인해 NPM과 다르게 추가적인 Scope가 필요해요.

  </Tab>
</Tabs>

es-toolkit의 함수를 사용하려면 아래와 같이 `import` 해서 사용하면 돼요.

```typescript
import { sum } from 'es-toolkit';

sum([1, 2, 3]);
```

## 브라우저

[jsdelivr](https://www.jsdelivr.com) 또는 [unpkg](https://unpkg.com) 같은 CDN에서 es-toolkit을 쓸 수 있어요. Lodash와 같이 `_` 변수에 모든 함수가 포함돼요.

```html
<script src="https://cdn.jsdelivr.net/npm/es-toolkit@^1"></script>
<script>
  var arr = _.chunk([1, 2, 3, 4, 5, 6], 3);
</script>
```

현대적인 브라우저에서는 [esm.sh](https://esm.sh)도 쓸 수 있어요.

```html
<script type="importmap">
  {
    "imports": {
      "es-toolkit": "https://esm.sh/es-toolkit@^1"
    }
  }
</script>
<script type="module">
  import { chunk } from 'es-toolkit';

  chunk([1, 2, 3, 4, 5, 6], 3);
</script>
```
