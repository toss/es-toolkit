# es-toolkit

A modern JavaScript utility library that's 2-3 times faster than lodash with a much smaller bundle size.

## Features

- **Performance**: 2-3x faster than lodash
- **Bundle Size**: Up to 97% smaller
- **TypeScript**: First-class TypeScript support
- **Tree Shaking**: Full support for tree shaking
