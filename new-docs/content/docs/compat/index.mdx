# es-toolkit/compat

`es-toolkit/compat` provides 100% Lodash compatibility while maintaining the performance benefits of es-toolkit.

## Installation

```bash
import { chunk } from 'es-toolkit/compat';
```

## Why compat?

- **Drop-in replacement**: Fully compatible with Lodash API
- **Better performance**: Still faster than original Lodash
- **Smaller bundle**: Significantly smaller than Lodash
- **Easy migration**: Migrate from Lodash gradually
